# Hola, soy Christian 👋

Fundador de **BData**, consultora especializada en **herramientas de gestión para la agroindustria chilena**. Diseño y desarrollo aplicaciones web que ordenan la operación de empresas agrícolas: inventarios, rendiciones de gastos, optimización de insumos y más.

🌱 Convierto procesos manuales y planillas dispersas en sistemas simples, trazables y útiles para decidir.

- **Frontend / Apps:** TypeScript, Next.js, React
- **Backend / Datos:** Supabase, PostgreSQL, Python
- **Deploy:** Netlify, Vercel
- **Enfoque:** gestión agroindustrial, automatización y análisis de datos

- **🌟 Agro BData** — Plataforma insignia de BData: gestión integral para empresas agroindustriales, pensada para ordenar la operación y apoyar la toma de decisiones.
- **Inventarios** — Aplicación web para control de inventarios (Next.js + Supabase).
- **Rendiciones** — Sistema de rendición de gastos para equipos en terreno.
- **Fertilizantes** — Optimizador de aplicación de fertilizantes.
